# 🚗 BMW Dealer Website

Профессиональная веб-платформа для официального дилерского центра BMW. Сайт сочетает в себе премиальную эстетику, современные анимации и мощный функционал для управления заявками и каталогом автомобилей.

## ✨ Основные особенности

- **Визуальное совершенство:** Плавные анимации с использованием **GSAP** и **Lenis**, а также интерактивные 3D-элементы на базе **Three.js**.
- **Полный клиентский путь:** От просмотра каталога до записи на сервис, тест-драйв и оформления лизинга/кредита.
- **Интерактивный каталог:** Фильтрация автомобилей по состоянию (новые/с пробегом) и типу двигателя.
- **Личный кабинет:** Система регистрации и авторизации пользователей для отслеживания их заявок.
- **Панель администратора:** Защищенный интерфейс для управления всеми входящими заявками и редактирования каталога автомобилей.

## 🛠 Технологический стек

### Frontend
- **HTML5 & Vanilla CSS3:** Кастомная архитектура стилей с акцентом на современный UI.
- **JavaScript (ES6+):** Чистая логика без тяжелых фреймворков для максимальной скорости.
- **GSAP:** Библиотека для высокопроизводительных анимаций и scroll-эффектов.
- **Three.js:** Интерактивная 3D графика (логотипы, визуализации).
- **Lenis:** Плавный инерционный скролл.

### Backend
- **Node.js & Express:** Быстрый и масштабируемый серверный движок.
- **SQLite:** Легкая и надежная реляционная база данных.
- **Body-parser & CORS:** Middleware для обработки запросов и кросс-доменного взаимодействия.

## 🚀 Установка и запуск

### Предварительные требования
- Установленный **Node.js** (рекомендуемая версия 16+)

### Шаги установки

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/eLbiii/bmw-dealer.git
   cd bmw-dealer
   ```

2. Установите зависимости:
   ```bash
   npm install
   ```

3. Запустите сервер:
   ```bash
   npm start
   ```

4. Откройте сайт:
   Приложение будет доступно по адресу `http://localhost:3000`.

## 🛡 Администрирование

Для доступа к панели администратора используйте страницу `/pages/login.html` (или специальный роут администратора, если предусмотрено). Стандартные учетные данные администратора находятся в конфигурации сервера (рекомендуется сменить перед развертыванием).
